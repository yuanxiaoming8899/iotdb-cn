<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">IoTDB（Internet of Things Database）是时序数据的数据管理系统，为用户提供数据采集、存储、分析等特定服务。</font><font style="vertical-align: inherit;">IoTDB凭借其轻量级结构、高性能和易用性，以及与Hadoop和Spark生态的无缝集成，满足了工业物联网领域海量数据存储、高吞吐量数据输入和复杂数据分析的需求。</font></font></p>
<h1 tabindex="-1" dir="auto"><a id="user-content-main-features" class="anchor" aria-hidden="true" tabindex="-1" href="#main-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">主要特点</font></font></h1>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">IoTDB的主要特点如下：</font></font></p>
<ol dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">灵活的部署策略。</font><font style="vertical-align: inherit;">IoTDB为用户提供了云平台和终端设备上的一键安装工具，以及桥接云平台和终端设备上数据的数据同步工具。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">硬件成本低。</font><font style="vertical-align: inherit;">IoTDB可以达到磁盘存储的高压缩比。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">高效的目录结构。</font><font style="vertical-align: inherit;">IoTDB支持智能网络设备复杂时序数据结构的高效组织、同类型设备时序数据的组织以及海量复杂时序数据目录的模糊搜索策略。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">高吞吐量读取和写入。</font><font style="vertical-align: inherit;">IoTDB支持数百万低功耗设备的强连接数据访问，以及上述智能联网设备和混合设备的高速数据读写。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">丰富的查询语义。</font><font style="vertical-align: inherit;">IoTDB支持跨设备和测量的时间序列数据的时间对齐、时间序列领域的计算（频域变换）以及时间维度上丰富的聚合功能支持。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">易于上手。</font><font style="vertical-align: inherit;">IoTDB支持SQL-Like语言、JDBC标准API和易于使用的导入/导出工具。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">与最先进的开源生态系统无缝集成。</font><font style="vertical-align: inherit;">IoTDB支持Hadoop、Spark等分析生态系统和Grafana等可视化工具。</font></font></li>
</ol>
